# Structure Explorer

Add a top 1% marketer to your team

Module 7

Structure Explorer shows you the distribution of URLs based on specific categories.

Click on any folder to see the breakdown of the URL distribution for each subfolder.

Hover over any path name to apply that folder as a segment.

Below are some actionable ways to use this report.

Click on the Depth filter and switch to the Data table view.

You’ll see the number of URLs by depth, which is the distance from one of the seed points.
