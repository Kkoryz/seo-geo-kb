# Link Intersect

Add a top 1% marketer to your team

Module 10

Link Intersect is our competitive analysis tool that lets you see domains and pages which link to your competitors’ websites, but not yours.

Here are some actionable use cases for it.

Let's say that we want to build links to our free AI writing tools. A perfect place to do that would be in listicles that mention other similar tools like ChatGPT, Jasper, and Copy.ai.

So, open the Competitive Analysis tool and select “referring pages”. Then:

Enter our page in the top section

Enter some competitors’ pages in the bottom section

Set all URLs to Exact URL mode since listicles mentioning tools usually link to homepages


Then hit Show link opportunities. This shows us all the referring pages that link to any of these homepages, but not to ours.

Not all of them will be relevant. To narrow them down, click on the Links to competitors dropdown and select All competitors.

To narrow in on just pages coming from authoritative websites, use the filters at the top. Select Do-follow and filter for DR: From 50. Then hit Show results.

This will give us a short list of pages that could potentially link back to us.

Let’s say that we want to build links for our fictional plumbing website. To do this, we first need to search on Google for plumbers in our city.

Next, open the Competitive Analysis tool and select “referring domains”.

Enter our website in the top section

Enter some top competitors’ domains in the bottom section

Set all URLs to Exact URL mode since citation links usually point to the homepage of local service providers


Then, hit Show link opportunities. This shows us a list of domains, many of which are clearly directories where you can add your company’s details.
