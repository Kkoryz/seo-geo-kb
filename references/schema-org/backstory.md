For an

[Article](/Article), typically a[NewsArticle](/NewsArticle), the backstory property provides a textual summary giving a brief explanation of why and how an article was created. In a journalistic setting this could include information about reporting process, methods, interviews, data sources, etc.### Values expected to be one of these types
