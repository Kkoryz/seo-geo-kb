# pageStart

A Schema.org Property

- Canonical URL: https://schema.org/pageStart
- Equivalent Property: bibo:pageStart
[Check for open issues.](https://github.com/schemaorg/schemaorg/issues?q=is%3Aissue+is%3Aopen+pageStart)

The page on which the work starts; for example "135" or "xiii".
