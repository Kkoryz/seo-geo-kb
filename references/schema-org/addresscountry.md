# addressCountry

A Schema.org Property

- Canonical URL: https://schema.org/addressCountry
[Check for open issues.](https://github.com/schemaorg/schemaorg/issues?q=is%3Aissue+is%3Aopen+addressCountry)

The country. Recommended to be in 2-letter

[ISO 3166-1 alpha-2](http://en.wikipedia.org/wiki/ISO_3166-1)format, for example "US". For backward compatibility, a 3-letter[ISO 3166-1 alpha-3](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-3)country code such as "SGP" or a full country name such as "Singapore" can also be used.### Used on these types
