# seatingCapacity

- Canonical URL: https://schema.org/seatingCapacity
[Check for open issues.](https://github.com/schemaorg/schemaorg/issues?q=is%3Aissue+is%3Aopen+seatingCapacity)

The number of persons that can be seated (e.g. in a vehicle), both in terms of the physical space available, and in terms of limitations set by law.


Typical unit code(s): C62 for persons.
