# returnPolicyCountry

[Check for open issues.](https://github.com/schemaorg/schemaorg/issues?q=is%3Aissue+is%3Aopen+returnPolicyCountry)

The country where the product has to be sent to for returns, for example "Ireland" using the

[name](/name)property of[Country](/Country). You can also provide the two-letter[ISO 3166-1 alpha-2 country code](http://en.wikipedia.org/wiki/ISO_3166-1). Note that this can be different from the country where the product was originally shipped from or sent to.### Used on these types
