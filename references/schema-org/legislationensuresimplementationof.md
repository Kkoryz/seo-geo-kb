# legislationEnsuresImplementationOf

A Schema.org Property

- Canonical URL: https://schema.org/legislationEnsuresImplementationOf
- Equivalent Property: eli:ensures_implementation_of
[Check for open issues.](https://github.com/schemaorg/schemaorg/issues?q=is%3Aissue+is%3Aopen+legislationEnsuresImplementationOf)

Indicates that this Legislation ensures the implementation of another Legislation, for example by modifying national legislations so that they do not contradict to an EU regulation or decision. This implies a legal meaning. Transpositions of EU Directive should be captured with

[legislationTransposes](/legislationTransposes).### Values expected to be one of these types
