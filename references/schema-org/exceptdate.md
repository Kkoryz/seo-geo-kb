# exceptDate

# exceptDate

A Schema.org Property

*This term is in the "new" area - implementation feedback and adoption from applications and websites can help improve our definitions.*

- Canonical URL: https://schema.org/exceptDate
[Check for open issues.](https://github.com/schemaorg/schemaorg/issues?q=is%3Aissue+is%3Aopen+exceptDate)

Defines a

[Date](/Date)or[DateTime](/DateTime)during which a scheduled[Event](/Event)will not take place. The property allows exceptions to a[Schedule](/Schedule)to be specified. If an exception is specified as a[DateTime](/DateTime)then only the event that would have started at that specific date and time should be excluded from the schedule. If an exception is specified as a[Date](/Date)then any event that is scheduled for that 24 hour period should be excluded from the schedule. This allows a whole day to be excluded from the schedule without having to itemise every scheduled event.### Used on these types
