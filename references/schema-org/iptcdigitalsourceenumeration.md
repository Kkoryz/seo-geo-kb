[IPTC](https://www.iptc.org/)"Digital Source" codes for use with the

[digitalSourceType](/digitalSourceType)property, providing information about the source for a digital media object. In general these codes are not declared here to be mutually exclusive, although some combinations would be contradictory if applied simultaneously, or might be considered mutually incompatible by upstream maintainers of the definitions. See the IPTC

[documentation](https://www.iptc.org/std/photometadata/documentation/userguide/)for

[detailed definitions](https://cv.iptc.org/newscodes/digitalsourcetype/)of all terms.
