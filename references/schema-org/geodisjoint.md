# geoDisjoint

A Schema.org Property

- Canonical URL: https://schema.org/geoDisjoint
[Check for open issues.](https://github.com/schemaorg/schemaorg/issues?q=is%3Aissue+is%3Aopen+geoDisjoint)

Represents spatial relations in which two geometries (or the places they represent) are topologically disjoint: "they have no point in common. They form a set of disconnected geometries." (A symmetric relationship, as defined in

[DE-9IM](https://en.wikipedia.org/wiki/DE-9IM).)### Values expected to be one of these types
