# QualitativeValue

A Schema.org Enumeration Type

- Canonical URL: https://schema.org/QualitativeValue
[Check for open issues.](https://github.com/schemaorg/schemaorg/issues?q=is%3Aissue+is%3Aopen+QualitativeValue)

A predefined value for a product characteristic, e.g. the power cord plug type 'US' or the garment sizes 'S', 'M', 'L', and 'XL'.
