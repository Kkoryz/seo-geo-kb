# gtin13

A Schema.org Property

- Canonical URL: https://schema.org/gtin13
[Check for open issues.](https://github.com/schemaorg/schemaorg/issues?q=is%3Aissue+is%3Aopen+gtin13)

The GTIN-13 code of the product, or the product to which the offer refers. This is equivalent to 13-digit ISBN codes and EAN UCC-13. Former 12-digit UPC codes can be converted into a GTIN-13 code by simply adding a preceding zero. See

[GS1 GTIN Summary](http://www.gs1.org/barcodes/technical/idkeys/gtin)for more details.### Values expected to be one of these types
