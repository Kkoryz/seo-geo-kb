# Third-party programs | Google Search Central Blog | Google for Developers

We've just updated the list of third-party programs that you can use to create Sitemaps. You can
check out the current list on
code.google.com.
That page also lists the Sitemaps Google Group for each language we support.

If you've written a tool that supports Google Sitemaps, let us know at
code+sitemaps@google.com.

The Sitemaps team is thankful for our great user community and appreciates all the work that's
gone into these tools.
