# Sunsetting the generic rich results search appearance on Search Console | Google Search Central Blog | Google for Developers

Sunsetting the generic rich results search appearance on Search Console

Tuesday, May 25, 2021

Over the last few years, we have been hard at work adding more and more specific rich results search appearances to the Search Console Performance
reports. Today we have 19 supported search
appearance types to help website owners understand how their websites perform in Google Search.

A few years ago we created a generic "Rich results" search appearance group that included multiple rich result types, including those that didn't
have their own breakout row on Search Console. This is not required anymore, since we now have breakouts for most rich result types, for example:
Event, FAQ,
HowTo, and more.

In addition, we realize that grouping data for rich results is not ideal, as each type may have a significantly different behavior. Therefore we decided to
sunset the Rich result search appearance.

We’ll keep investing in adding more rich result types to Search Console, and hope the reports will continue being a source of insights for you.
