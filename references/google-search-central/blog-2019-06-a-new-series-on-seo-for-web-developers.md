# A video series on SEO myths for web developers | Google Search Central Blog | Google for Developers

We invited members of the SEO and web developer community to join us for a new video series called "SEO mythbusting".

In this series, we discuss various topics around SEO from a developer's perspective, how we can
work to make the "SEO black box" more transparent, and what technical SEO might look like as the
web keeps evolving. We already published a few episodes:
