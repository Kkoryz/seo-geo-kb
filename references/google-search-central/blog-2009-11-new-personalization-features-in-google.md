# New personalization features in Google Friend Connect | Google Search Central Blog | Google for Developers

New personalization features in Google Friend Connect

Wednesday, November 04, 2009

Just a few weeks ago, we made
Google Friend Connect
a lot easier to use by dramatically simplifying the setup process. Today, we're excited to
announce several new features that make it possible for website owners to get to know their
users, encourage users to get to know each other, and match their site content (including Google
ads) to visitors' interests.
