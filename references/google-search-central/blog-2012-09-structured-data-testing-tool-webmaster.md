# Structured Data Testing Tool | Google Search Central Blog | Google for Developers

Today we're excited to share the launch of a shiny new version of the rich snippet testing tool,
now called the
structured data testing tool.
The major improvements are:

We've improved how we display rich snippets in the testing tool to better match how they appear
in search results.

The brand new visual design makes it clearer what structured data we can extract from the page,
and how that may be shown in our search results.

The tool is now available in languages other than English to help webmasters from around the
world build structured-data-enabled websites.
