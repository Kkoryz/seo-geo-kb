# Introducing the Search Analytics API | Google Search Central Blog | Google for Developers

Wednesday, August 05, 2015

With the great feedback from the
[Search Analytics](https://support.google.com/webmasters/answer/6155685)
feature in Google Search Console, we've decided to make this data accessible for developers via
API. We hope that the
[Search Analytics API](/webmaster-tools/v3/searchanalytics) will help
you to bake search performance data into your apps and tools.

If you've used any of Google's other APIs, or maybe one of the existing Search Console APIs, then
getting started will be easy! The
[how-to page](/webmaster-tools/v3/how-tos/search_analytics) has
examples in Python that you can use as recipes for your own programs. For example, you can use the
API to:

-
[Verify the presence of data](/webmaster-tools/v3/how-tos/search_analytics#verify--the-presence-of-data)(what's the most recent date you can request?) -
[Top 10 queries by click count](/webmaster-tools/v3/how-tos/search_analytics#top-10-queries-sorted-by-click-count-descending) -
[Top 10 pages](/webmaster-tools/v3/how-tos/search_analytics#top-10-pages-sorted-by-click-count-descending) -
[Top 10 queries in India](/webmaster-tools/v3/how-tos/search_analytics#top-10-queries-in-india-sorted-by-click-count-descending) -
[Top 10 mobile queries in India](/webmaster-tools/v3/how-tos/search_analytics#top-10-mobile-queries-in-india-sorted-by-click-count-descending)

What will you cook up with the new API? We're curious to see how new tools and apps that use this
API will satisfy the hunger for even more information about your site's performance in Google
Search! If you've integrated this API into a tool, we'd love to hear about it in the comments. If
you've run into any questions about the API, you can drop by our
[webmaster help forum](https://support.google.com/webmasters/go/community).
