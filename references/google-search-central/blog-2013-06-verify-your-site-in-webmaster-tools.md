# Verify your site in Webmaster Tools using Google Tag Manager | Google Search Central Blog | Google for Developers

On the Webmaster Tools home page, click Manage site for the site you'd like
to verify, then select Verify this site. If you haven't added the site yet,
you can click the Add a site button in the top right corner.
